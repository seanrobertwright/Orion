<task type="auto">
  <name>Task 2: Set up PostgreSQL with Docker and Drizzle configuration</name>
  <files>
    docker-compose.yml
    drizzle.config.ts
    src/db/index.ts
  </files>
  <action>
    Create docker-compose.yml:
    ```yaml
    version: '3.8'
    services:
      db:
        image: postgres:16-alpine
        container_name: orion-db
        environment:
          POSTGRES_USER: postgres
          POSTGRES_PASSWORD: postgres
          POSTGRES_DB: orion
        ports:
          - "5432:5432"
        volumes:
          - orion_pgdata:/var/lib/postgresql/data
    volumes:
      orion_pgdata:
    ```

    Create drizzle.config.ts:
    ```typescript
    import { defineConfig } from 'drizzle-kit';

    export default defineConfig({
      schema: './src/db/schema.ts',
      out: './drizzle',
      dialect: 'postgresql',
      dbCredentials: {
        url: process.env.DATABASE_URL!,
      },
    });
    ```

    Create src/db/index.ts:
    ```typescript
    import { drizzle } from 'drizzle-orm/postgres-js';
    import postgres from 'postgres';
    import * as schema from './schema';

    const connectionString = process.env.DATABASE_URL!;
    const client = postgres(connectionString);
    export const db = drizzle(client, { schema });
    ```

    Add scripts to package.json:
    - "db:generate": "drizzle-kit generate"
    - "db:migrate": "drizzle-kit migrate"
    - "db:push": "drizzle-kit push"
    - "db:studio": "drizzle-kit studio"
  </action>
  <verify>
    docker-compose up -d starts PostgreSQL
    psql -h localhost -U postgres -d orion connects successfully
  </verify>
  <done>
    PostgreSQL running in Docker, Drizzle configured and ready for schema
  </done>
</task>

<task type="auto">
  <name>Task 3: Create complete database schema for Phase 1</name>
  <files>
    src/db/schema.ts
  </files>
  <action>
    Create src/db/schema.ts with all Phase 1 models:

    **users table** (for NextAuth + profile):
    - id: text (primary key, cuid)
    - name: text
    - email: text (unique)
    - emailVerified: timestamp
    - image: text
    - salaryMin: integer (nullable)
    - salaryMax: integer (nullable)
    - workPreference: text (remote/hybrid/onsite, nullable)
    - questionnaireCompleted: boolean (default false)
    - createdAt: timestamp (default now)
    - updatedAt: timestamp

    **accounts table** (NextAuth OAuth):
    - Standard NextAuth fields for OAuth provider linking

    **sessions table** (NextAuth):
    - Standard NextAuth session fields

    **jobs table**:
    - id: text (primary key, cuid)
    - userId: text (foreign key to users)
    - title: text (required - per CONTEXT.md validation)
    - company: text (required - per CONTEXT.md validation)
    - location: text (nullable)
    - url: text (nullable)
    - description: text (nullable)
    - requirements: text (nullable)
    - niceToHaves: text (nullable)
    - createdAt: timestamp
    - updatedAt: timestamp

    **applications table**:
    - id: text (primary key, cuid)
    - userId: text (foreign key to users)
    - jobId: text (foreign key to jobs)
    - status: text (enum: saved, applied, interviewing, offered, rejected)
    - appliedAt: timestamp (nullable)
    - notes: text (nullable)
    - coverLetterId: text (foreign key to coverLetters, nullable)
    - createdAt: timestamp
    - updatedAt: timestamp

    **statusHistory table** (per CONTEXT.md - silent logging):
    - id: text (primary key, cuid)
    - applicationId: text (foreign key to applications)
    - fromStatus: text (nullable for initial)
    - toStatus: text
    - changedAt: timestamp (default now)

    **interviews table**:
    - id: text (primary key, cuid)
    - applicationId: text (foreign key to applications)
    - scheduledAt: timestamp
    - type: text (phone, video, onsite, technical)
    - notes: text (nullable)
    - questionsAsked: text (nullable)
    - companyFeedback: text (nullable)
    - outcome: text (enum: pending, passed, advanced, rejected, nullable)
    - reminderSent: boolean (default false)
    - createdAt: timestamp
    - updatedAt: timestamp

    **coverLetters table**:
    - id: text (primary key, cuid)
    - userId: text (foreign key to users)
    - name: text
    - content: text
    - createdAt: timestamp
    - updatedAt: timestamp

    **userQuestionnaire table**:
    - id: text (primary key, cuid)
    - userId: text (foreign key to users, unique)
    - technicalSkills: jsonb (array of skills)
    - preferredRoles: jsonb (array of role types)
    - preferredIndustries: jsonb (array)
    - yearsExperience: integer (nullable)
    - careerGoals: text (nullable)
    - completedAt: timestamp (nullable)
    - createdAt: timestamp
    - updatedAt: timestamp

    **jobFeedback table** (for LEARN-02):
    - id: text (primary key, cuid)
    - userId: text (foreign key to users)
    - jobId: text (foreign key to jobs)
    - interested: boolean
    - reason: text (nullable)
    - createdAt: timestamp

    Export all tables and define relations using Drizzle relations().
  </action>
